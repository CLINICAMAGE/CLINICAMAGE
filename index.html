Backup e Exportação

Proteja seus dados — exporte para mídia física ou nuvem a qualquer momento

☁️ Sobre seus dados
Na versão atual (HTML local), todos os dados ficam salvos no localStorage deste navegador.
Na versão web em nuvem (próxima etapa), os dados migrarão para o Supabase com backup automático diário.

Recomendação: exporte um backup completo mensalmente e salve em local seguro.
📊 Status atual
📦 Exportar Dados
🔄 Importar / Restaurar
⚠️ Importar um backup substitui todos os dados atuais. Faça um backup antes de restaurar.
☁️ Versão Web em Nuvem
A versão web utiliza Supabase (PostgreSQL) com:
• Backup automático diário
• Histórico de 7–30 dias
• Exportação SQL completa
• Crescimento ilimitado por plano
• Portabilidade total dos dados
🚀 Quando estiver pronto para migrar para a nuvem, todos os dados exportados aqui podem ser importados para o banco web sem perda alguma.
